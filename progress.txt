【v2.0 大版本更新 - 2026-05-14】
===================================================================

本次更新根据《核心玩法设计说明书》+《建筑+道具详细设计说明书》
进行全面重构优化，核心改动如下：

[已完成] 1. 道具体系重构（聚焦4档通用道具）
   - 1级 陶罐（普通）— 基础产出
   - 2级 油纸伞（精良）— 2陶罐合成
   - 3级 折扇/刺绣（珍稀）— 2油纸伞合成
   - 4级 玉佩（传世）— 2珍稀合成
   - 建材：初级/中级/高级

[已完成] 2. 新增古镇建造系统（8种建筑 × 3级）
   市井街：民居、杂货铺、茶馆、驿站
   风雅巷：书斋、绣坊、画院、棋社
   每建筑1/3/5级外观差异明显，离线收益递增

[已完成] 3. 新增离线挂机系统
   - 退出后自动计算离线收益（最多8小时）
   - 上线弹窗一键领取
   - 支持看广告双倍领取

[已完成] 4. 新增每日签到系统
   - 7日签到奖励梯度设计
   - 第3天送稀有道具，第7天送限定皮肤

[已完成] 5. 新增每日任务系统
   - 完成3次合成
   - 交付2笔订单
   - 解锁1个建筑
   奖励：金币 + 建材

[已完成] 6. 新增新手引导流程
   - 0-10秒：引导首次合成
   - 0-3分钟：走完合成→订单→建造完整循环
   - 4步引导逐步推进

[已完成] 7. 合成特效系统
   - 各档道具不同合成特效
   - 1级：简单闪光 + "合成成功"
   - 2级：金色闪光 + "合成精良油纸伞" + 伞面展开
   - 3级：金色光芒 + 墨晕效果 + "合成珍稀折扇"
   - 4级：耀眼金光 + 玉佩旋转 + "合成传世玉佩！" + 弹窗恭喜

[已完成] 8. 核心三层玩法闭环
   主玩法闭环：合成 → 交付订单 → 获得金币+建材 → 解锁建筑 → 离线收益提升
   成长闭环：古镇升级 → 解锁新街区 → 开放更多建筑位 → 挂机收益倍率提升
   留存闭环：每日签到 → 完成任务 → 离线挂机收收益 → 集齐图鉴领奖励

[已完成] 9. 后端API同步更新
   - 适配新的建筑/建材/离线收益数据结构
   - 新增签到/任务相关接口

【v3.0 重大重构 - 2026-05-14】解决"没法玩"问题
===================================================================

核心问题诊断：
1. v1.0/v2.0 没有主线目标 → 玩家不知道要做什么
2. 没有剧情引导 → 进入游戏就是空白棋盘
3. 操作没有爽感 → 合成没有特效和反馈
4. 画面太简陋 → 不像游戏，像原型
5. 没有收集成就感 → 没有进度可视化

v3.0 核心改进：

[已完成] 1. 主线剧情驱动（GDD原文设定）
   - 开场剧情：沈绾发现残缺《山河锦绣图》
   - 主线任务：收集7大景点残卷 → 复原锦绣图
   - 8个递进任务引导玩家每一步
   - 任务完成弹窗奖励反馈

[已完成] 2. 精美Q萌画质（借鉴时光大爆炸）
   - 全新标题画面：Q版沈绾+江南水乡背景
   - 动态背景浮动效果
   - 脉冲发光开始按钮
   - 剧情对话系统：头像+对话框+打字效果

[已完成] 3. 合成爽感系统
   - 4档合成特效（Lv1-Lv4递增）
   - 粒子爆发动画
   - 浮动文字恭喜
   - 选中放大+发光边框

[已完成] 4. 场景修缮系统
   - 7大景点可视化进度
   - 每景点3阶段修缮
   - 修缮奖励声望
   - 声望解锁新场景

[已完成] 5. 订单任务系统
   - 4个动态订单
   - 道具数量实时检测
   - 一键交付+奖励
   - 订单自动刷新

[已完成] 6. NPC对话系统
   - 苏轼/李白/戚继光3位名人
   - 专属台词+emoji头像
   - 弹窗对话交互

[已完成] 7. 山河图卷收集
   - 可视化收集进度
   - 7大景点完成度
   - 图卷浮动动画

[已完成] 8. 全局优化
