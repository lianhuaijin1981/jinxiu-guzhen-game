# 《锦绣古镇：汉服时代》v3.0 融合设计文档

## 融合定位

**融合方向**：引入《时光大爆炸》的"时代晋升+名士养成+科技树+征战+奇观"等核心系统，与《锦绣古镇》的"汉服织造+国风Merge-2合成+古镇建造"深度融合，打造一款以**汉服文明演变**为主线的文明超进化模拟经营游戏。

**融合口号**：从一间小绣坊到皇家织造局，见证千年汉服文明的锦绣传奇。

### 1.1 时代设定（7大汉服时代）

将原版的"7大地标场景"重构为"7大汉服时代"，玩家需要完成每个时代的全部目标才能晋升：

| 时代 | 名称 | 主题色 | 核心解锁 | 时代目标 |

| 2 | **周礼时代** | 青铜色 | 深衣形制、礼仪汉服 | 解锁名士系统 |
| 3 | **秦汉时代** | 玄黑色 | 曲裾、直裾、铠甲外袍 | 解锁战斗（织锦大赛） |

## 第三部分：科技研究系统（引入时光大爆炸科技树）

### 3.1 科技树五大分支

将时光大爆炸的科技树重构为汉服织造主题的五大科技线：

**分支一：养蚕缫丝术**
- 野蚕驯化 → 家蚕培育 → 多季蚕 → 良种蚕
- 效果：提升蚕茧产出量和品质

**分支二：纺织工艺**
- 手工纺纱 → 纺车改良 → 水力纺机 → 提花机
- 效果：解锁新合成配方，提升合成效率

**分支三：染色印花术**
- 植物染色 → 矿物颜料 → 扎染工艺 → 缂丝技艺
- 效果：解锁彩色织物，提升道具品质

**分支四：刺绣工艺**
- 平针绣 → 盘金绣 → 双面绣 → 精微绣
- 效果：提升高阶道具合成成功率

**分支五：裁剪形制**
- 深衣制 → 襦裙制 → 马面裙 → 龙袍制
- 效果：解锁新汉服款式，提升订单收益

### 3.2 研究规则

- 每次只能研究一种技术
- 可使用加速卡（玉璧）加速
- 技术研究 + 精研研究可同时进行
- 精研有时代压制效果（高时代对低时代有加成）
- 完成一个时代的全部科技研究可获得"时代精通"奖励

## 第四部分：战斗系统重构（织锦大赛）

### 4.1 织锦大赛（替代时光大爆炸的征战）

将战斗改为**织锦大赛/比美大赛**形式，贴合汉服世界观：

**赛制规则**：
- 每回合双方各派一名名士上阵
- 名士释放"织锦绝技"（如：缂丝绝技、盘金绝技、双面绣绝技）
- 比拼"织艺值"高低决定胜负
- 胜出方获得织锦积分和布料奖励

**织锦大赛类型**：
1. **主题比美赛**：系统出题（如"春日宴"主题），双方搭配汉服套装比拼
2. **技艺对决**：纯拼名士织艺值，一决高下
3. **限时刺绣赛**：限定时间内合成指定数量高阶道具
4. **全服锦绣榜**：赛季制排行榜，奖励限定称号和汉服皮肤

### 4.2 兵种 → 织娘编队

将时光大爆炸的兵种系统改为织娘编队：

| 织娘类型 | 定位 | 克制关系 |
|---------|------|---------|
| **蚕娘** | 前排（坦克） | 被绣娘克制，克制染娘 |
| **织女** | 中排（输出） | 被染娘克制，克制蚕娘 |
| **染娘** | 后排（辅助） | 被蚕娘克制，克制织女 |
| **绣娘** | 特殊（控制） | 被织娘克制，克制蚕娘 |
| **裁娘** | 治疗（恢复） | 无克制关系 |

### 4.3 远征与巡视

- **丝路远征**：派遣名士队伍沿丝绸之路远征，带回西域织物和稀有材料
- **江南巡视**：巡视江南各地，解锁新织坊和特殊事件
- **宫廷选秀**：将织造的汉服送入宫中，获得皇家赏赐
